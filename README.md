Este projeto é um sistema simples de CRUD (Create, Read, Update, Delete) para gerenciamento de clientes, desenvolvido com HTML, CSS e JavaScript.

A aplicação permite:

Cadastrar novos clientes através de um modal

Listar clientes em uma tabela dinâmica

Editar informações existentes

Excluir registros

Validar campos antes de salvar

Atualização automática da tabela após cada ação

O layout utiliza arquivos CSS separados (main.css, button.css, records.css, modal.css) e toda a lógica funcional é controlada por main.js.

Ideal para estudos de manipulação do DOM e armazenamento de dados em aplicações Web simples.
